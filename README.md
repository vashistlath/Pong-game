# Python Pong

I made this as one of my first Python projects while learning game development with the turtle library. It is a simple two-player Pong game where each player controls a paddle and they try to stop the ball from getting past them.

## Features

* Two-player Pong
* Score tracking
* Paddle and wall collision
* Ball resets after every point

## Controls

**Player 1**

* `W` - Up
* `S` - Down

**Player 2**

* `↑` - Up
* `↓` - Down

## Running the game

Make sure you have Python installed, then just run:

```bash
python main.py
```
